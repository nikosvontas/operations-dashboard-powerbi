# Operations Performance Dashboard – Certification Project

## Project Overview
This dashboard was developed as part of the DavidsonX Power BI certification coursework at the edx platform using Microsoft Power BI.

## Tools Used
- Microsoft Power BI
- Power Query
- Basic DAX calculations

## Data Source
Public repository datasets provided during certification training.

## KPIs Analyzed
-Growth rate

## Purpose
The project demonstrates fundamental skills in dashboard development, data transformation, and operational data visualization.

## Key Learning Outcome
- Building KPI dashboards  
- Data cleaning and transformation  
- Basic data modeling  
- Translating datasets into business insights
